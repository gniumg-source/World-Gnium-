# World Gnium™

World Gnium es un proyecto creado por gniumg-source. Este repositorio contiene el scaffold inicial para el proyecto World Gnium — documentación, carpeta de código fuente y un workflow CI básico.

## Quick start

1. Clona el repositorio:
   ```bash
   git clone https://github.com/gniumg-source/world-gnium.git
   cd world-gnium
   ```

2. Revisa `src/` para código y `docs/` para la documentación.
